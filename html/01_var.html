Using var

When a variable is declared using the keyword var, it may be changed later. This was initially the only method for declaring varibles in JavaScript, and must be used if you want your code to run on older browsers. Using var also lets you redeclare a variable, which is not possible with let or const. The syntax for declaring a variable with var is:

  • Declare variable carName: var carName;
  • Declare variable carName and assign the value "Volvo": var carName = "Volvo";
  • Declare variable x: var x;
  • Declare variable x and assign the value 5: var x = 5;
